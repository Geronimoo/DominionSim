DominionSim

A simulator for the Dominion card game

About

DominionSim is a simulator for the Dominion card game. It allows you to build and run simulations using Java and Maven.

Prerequisites

You will need the following to build and run the simulator:

  • Java SE SDK 1.8 or higher
  • Maven

Build

To build the simulator JAR file, run:

mvn package

Run

To run the simulator, use:

mvn exec:java
