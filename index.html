Dominion stuff

To run the simulator you'll need to configure a site exception for this website.

Launch Geronimoo's Dominion Simulator (only works if you have java web start set up correctly in your browser)

To launch the simulator client you need to have Java installed. Then you can start it up by clicking Windows button and typing 'javaws https://raw.githubusercontent.com/Geronimoo/DominionSim/master/assets/dominion.jnlp'. Once you've done that there will be an icon on your desktop to launch it next times.
